Class Morph_Property_Generic

Description
Description | Descendents | Vars (details) | Methods (details)

Class to represent a property within a table

See Morph_Property_* for more specific implementations

If you wish to create a more specialised property type you can extend this class. Overloading getValue() and setValue() allow you to provide automatic marshalling of data.
