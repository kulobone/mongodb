Class Morph_Query

Description
Description | Methods (details)

A query class to represent queries to the database

This class provides a uber fluent interface so you can do cool stuff like:

  1.  $query = new Morph_Query();
  2.  $query->limit(10)
  3.        ->property('name')
  4.        ->equals('jon')
  5.        ->property('age')
  6.        ->greaterThanOrEqualTo('30')
  7.        ->lessThan('40');
  8.  
  9.  $results = $MorphStorage->findByQuery($anObject, $query);

The above example is roughly eqivalent to the SQL query:

  1.  SELECT * FROM `$anObject->collection()` WHERE
  2.      `name` = 'jon'
  3.      AND `age` >= 30
  4.      AND `age` <  40
  5.  LIMIT 10;

Yeah, it looks like more code but with autocomplete it is super fast to type in!
